NoNew887 Page about making a bootloader.

This page is about making an x86 bootloader with Assembly language.


Before making a bootloader, we should know what does "bootloader" do. So, when you power on your PC, it starts in Real Mode (16-bit) and can only access 64 KB of RAM, even if you have more. The bootloader transitions from this state into an enviroment ready for the kernel.

This is an example of a simple x86 Assembly bootloader:


    jmp $

    times 510-($-$$) db 0
    db 0x55, 0xaa

To try this code, you need to save it in an ASM file, compile it using NASM (Netwide Assembler) and run it in QEMU. These programs support Windows and Linux.

If you named your ASM file "boot.asm", open a command line program (terminal), go to the folder where your file is and type this command:


    nasm -f bin boot.asm -o boot.bin

Now you have "boot.bin" file in the same folder. Let's run it in QEMU with this command:


    qemu-system-x86_64 -drive format=raw,file=boot.bin
