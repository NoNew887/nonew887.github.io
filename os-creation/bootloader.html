NoNew887 Page about making a bootloader.

This page is about making an x86 bootloader with Assembly language.


Before making a bootloader, we should know what does "bootloader" do. So, when you power on your PC, it starts in Real Mode (16-bit) and can only access 64 KB of RAM, even if you have more. The bootloader transitions from this state into an enviroment ready for the kernel.

This is an example of a simple x86 Assembly bootloader:


    jmp $

    times 510-($-$$) db 0
    db 0x55, 0xaa

To turn this assembly code into a bootable disk image, you can use NASM (Netwide Assembler) and run it inside an emulator like QEMU.
